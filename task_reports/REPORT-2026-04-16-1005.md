# Technical Task Report: MCP Source Status Persistence and Last-Check Metadata
**Timestamp**: 2026-04-16 10:05
**Task Objective**: Implement persistence for MCP source diagnostics results by adding metadata fields to the `MCPSource` model and updating the diagnostics flow to save results to the database.

---
## 1. Technical Implementation
### Model Changes
The `MCPSource` model in `tree_ui/models.py` was extended with the following fields:
- `last_checked_at`: Timestamp of the most recent diagnostic check.
- `last_check_ok`: Boolean indicating whether the check was successful.
- `last_check_label`: A brief status label (e.g., "Ready", "Unavailable").
- `last_check_message`: Detailed output or error message from the diagnostic check.
- `last_check_tool_count`: Number of tools discovered during the check.
- `last_check_tools_summary`: Comma-separated list of discovered tool names.

### Diagnostics Flow Update
- Added a `save_diagnostics_result` helper function in `tree_ui/services/mcp/source_status.py` to handle updating the model instance with diagnostic results.
- Modified the `mcp_source_test` view in `tree_ui/views.py` to call `save_diagnostics_result` after performing the diagnostic check. Results are now persisted in the database instead of being stored in the session.
- Updated the `mcp_source_list` view to retrieve the persisted diagnostic data from the database.

### UI Enhancements
- Updated `tree_ui/templates/tree_ui/mcp_source_list.html` to display the persisted last-check status, including the timestamp of the last check.

def save_diagnostics_result(source_model: Any, result: Dict[str, Any]) -> None:
    source_model.last_checked_at = timezone.now()
    source_model.last_check_ok = result.get("ok", False)
    source_model.last_check_label = result.get("label", "Unknown")
    source_model.last_check_message = result.get("message", "")
    source_model.last_check_tool_count = result.get("tool_count", 0)
    source_model.last_check_tools_summary = ", ".join(result.get("tool_names", []))
    source_model.save(update_fields=[...])
